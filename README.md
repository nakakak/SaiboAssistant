## v0.2.13 相对 v0.2.12 的改进

- 首次向导**不再预填**商家云地址，须按你的环境填写（如 `ws://IP:端口/bridge/connector`）。
- 修复首次配置完成后界面崩溃的问题。
- 听写默认仅 **inject**（不启用悬浮字幕）。
- 配置完成后程序在**菜单栏/系统托盘**常驻；关闭窗口只是隐藏，可从托盘打开「运行状态」。
