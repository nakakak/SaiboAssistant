# SaiboAssistant

**赛搏小小助手**（OpenClaw Connector）— 本仓库 **仅提供 Release 安装包下载**，不含业务源码。

## 快速开始

1. 打开 **[最新 Release](https://github.com/nakakak/SaiboAssistant/releases/latest)**，按系统下载 `SaiboAssistant-*` 单文件。
2. 阅读 **[下载与配置指南.md](./下载与配置指南.md)**，完成首次向导配置与连接。
