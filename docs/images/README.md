# 配置截图目录

把赛搏小小助手的配置界面截图放在本目录，供 [下载与配置指南.md](../../下载与配置指南.md) 引用。

## 推荐文件名（v0.2.15 图文包）

**请把 PNG/JPG 放在本目录** `SaiboAssistant/docs/images/`。

### 必传（当前 5 张）

| 保存为 | 你的截图内容 |
|--------|----------------|
| `03-first-run-welcome.png` | 首次向导 — 勾选 OpenClaw / 听写 |
| `04-wizard-server-mac-openclaw.png` | 首次向导 — 商家云 + **连接码或 MAC** + Gateway Token |
| `10-wizard-consent.png` | 首次向导 — 权限与同意 |
| `06-status-connected.png` | 运行状态 — **已连接商家云** |
| `09-admin-device-mac.png` | 管理台 — 设备管理中的 **MAC**（可选，因可用连接码） |

### 可选

| 文件名 | 内容 |
|--------|------|
| `01-download-release.png` | GitHub Release Assets 下载页 |
| `02-mac-gatekeeper.png` | Mac「无法打开」/ 仍要打开 |
| `07-settings-server-url.png` | 连接设置 — server_url |
| `08-tray-menu.png` | 菜单栏 / 托盘菜单 |

格式：**PNG** 或 **JPG**，宽度 800–1200px。

## 截图与文档约定

- 可打码：Token、内网 IP、完整 MAC。
- **v0.2.15+**：向导优先展示 **6 位连接码**；v0.2.14 截图仅 MAC。
- 助手 `server_url` 必须是 `.../bridge/connector`，不是管理台设备的 `.../ws`。
